# 🛡️ CedarGuard: Property Compliance & Risk Manager

**The Enterprise-Grade Single Source of Truth for High-Risk Building Compliance.**

CedarGuard is a mission-critical platform engineered for UK Social Housing Providers, Local Authorities, and senior housing executives. It provides a definitive, intelligence-led approach to managing complex property portfolios, ensuring absolute alignment with the **Building Safety Act 2022**, **Fire Safety Act 2021**, and the emerging requirements of **Awaab’s Law**.

---

## ✨ Strategic Intelligence

### 🔬 AI-Powered Compliance Discovery
- **Automated Mapping**: Intelligent discovery of regulatory obligations based on property taxonomy and building height (HRB status).
- **Evidence Validation**: Seamlessly map inspection reports and safety cases to specific statutory requirements.
- **Regulatory Autopilot**: Real-time alerts and briefings for managers on critical legal obligations.

### 📊 Executive Portfolio Oversight
- **RAG Status Intelligence**: Live visibility into portfolio-wide compliance health via interactive heatmaps and executive dashboards.
- **Advanced Risk Architecture**: Integrated RAID (Risks, Actions, Issues, Decisions) management with automated scoring (Likelihood x Impact).
- **Governance Vault**: Centralized, encrypted repository for mandatory safety signatures and audit-ready documentation.

### 📱 Premium UX & Mobile Resilience
- **State-of-the-Art Interface**: A fluid, glassmorphism-inspired UI designed for clarity and high-performance oversight.
- **On-Site Capability**: Fully responsive Progressive Web App (PWA) optimized for mobile capture during on-site inspections.
- **Accessibility First**: Built to meet UK public sector accessibility standards (WCAG 2.0 AA).

## 🚀 Deployment

### Prerequisites
- Node.js v18.0.0+
- Firebase Project for Data & Authentication
- Gemini AI API Credentials

### Installation
1. Clone the repository to your environment.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure your enterprise environment variables in `.env.local`.
4. Launch the developer console:
   ```bash
   npm run dev
   ```

### Production Build
To generate a production-optimized bundle:
```bash
npm run build
```
